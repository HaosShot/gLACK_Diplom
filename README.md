# GLACK — голосовой ИИ-ассистент для управления ПК

GLACK (GLACK Local Assistant Computer Kit) — это локальный голосовой ассистент для Windows, macOS и Linux, который управляет компьютером с помощью естественно-языковых команд на русском языке.

Проект использует локальные модели через [Ollama](https://ollama.com/) и распознавание речи через OpenAI Whisper, не отправляя данные пользователя в облако.

## Возможности

- **Голосовое и текстовое управление** — скажи команду или напиши её в интерфейсе.
- **Управление браузером** — открыть сайт, поиск в Google/YouTube, навигация по вкладкам.
- **Работа с приложениями** — запуск блокнота, калькулятора, терминала, проводника.
- **Управление окнами и рабочим столом** — свернуть, развернуть, переключить, закрыть окна.
- **Мышь и клавиатура** — клики, перемещение, печать текста, горячие клавиши.
- **Скриншоты и анализ экрана** — сделать скриншот и спросить vision-модель, что на экране.
- **Пользовательские команды** — добавь свои фразы для запуска программ, сайтов, текстов и горячих клавиш.

## Требования

- Python 3.10+
- [Ollama](https://ollama.com/download) установлена и запущена
- Микрофон (для голосовых команд)
- Windows / macOS / Linux

## Установка

1. Клонируй репозиторий:

```bash
git clone https://github.com/USERNAME/glack.git
cd glack
```

2. Установи зависимости:

```bash
pip install -r requirements.txt
```

3. Скачай необходимые модели в Ollama:

```bash
ollama pull qwen2.5:3b
ollama pull llava:7b
```

> `qwen2.5:3b` — основная модель для выбора инструментов.  
> `llava:7b` — модель для анализа скриншотов экрана.

## Запуск

```bash
python main.py
```

Откроется окно GLACK. Можно говорить в микрофон или писать команды в поле ввода.

## Примеры команд

| Команда | Действие |
|---------|----------|
| «Открой браузер» | Открывает Google |
| «Найди котиков в гугле» | Поиск в Google |
| «Включи первое видео phonk» | Открывает первое видео на YouTube |
| «Открой вк» | Открывает vk.com |
| «Сделай скриншот» | Сохраняет скриншот экрана |
| «Что на экране» | Анализирует экран через vision-модель |
| «Напиши привет мир» | Печатает текст в активное окно |
| «Нажми ctrl+s» | Отправляет горячую клавишу |
| «Сверни все окна» | Win+D / аналог |

## Пользовательские команды

Во вкладке **Команды** можно добавить свои фразы-триггеры:

- **launch** — запустить программу или файл
- **url** — открыть сайт
- **type** — напечатать текст
- **hotkey** — нажать комбинацию клавиш

Пример: фраза «включи майнкрафт» → launch → `D:\Minecraft\LL.exe`.

## Конфигурация

Основные параметры в `config.py`:

```python
WHISPER_MODEL  = "base"      # модель распознавания речи
LLM_MODEL      = "qwen2.5:3b" # основная языковая модель
VISION_MODEL   = "llava:7b"   # модель анализа изображений
RECORD_SECONDS = 5            # длительность записи голоса
LANGUAGE       = "ru"         # язык распознавания
```

## Безопасность и ограничения

- Все модели работают локально через Ollama.
- Ассистент может нажимать клавиши и управлять окнами — используй с осторожностью.
- Для прерывания скриптов PyAutoGUI установлен `FAILSAFE = False`; будь внимателен при тестировании.
